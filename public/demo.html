🚀 Product Orchestrator - Demo

Funksjonalitet

✅ GitHub Integrasjon: Kobler til GitHub API for å hente brukerens repositories
✅ Intelligent Analyse: Analyserer hvert repository og genererer anbefalinger basert på:
  • Dokumentasjon (README, beskrivelse, hjemmeside)
  • Aktivitet (siste oppdatering, åpne issues)
  • Synlighet (stjerner, forks)
  • Vedlikehold og kvalitet
✅ Markedsmuligheter: Hver anbefaling inkluderer informasjon om potensielle markedsmuligheter
✅ Filtreringsmuligheter:
  • Alle repositories
  • Markedsmuligheter
  • Trenger oppmerksomhet (høy prioritet)
  • Aktive repositories
  • Inaktive repositories
✅ Dashboard med statistikk:
  • Totalt antall repositories
  • Aktive repositories (siste 30 dager)
  • Totalt antall stjerner
  • Repositories som trenger oppmerksomhet

Hvordan bruke

Steg 1: Start backend serveren med npm run dev
Steg 2: Start frontend med npm run dev:client
Steg 3: Åpne http://localhost:3000 i nettleseren
Steg 4: Logg inn med GitHub Personal Access Token
Steg 5: Utforsk dine repositories og få anbefalinger!
